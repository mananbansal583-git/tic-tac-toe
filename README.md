# Tic-Tac-Toe (C++)

A simple 2-player console-based Tic-Tac-Toe game written in C++.

## Features

- 3x3 game board
- Two-player mode (X and O)
- Win detection (rows, columns, diagonals)
- Draw detection after 9 moves
- Turn switching between players

## How to Run

1. Compile the program:
   g++ tictactoe.cpp -o prog

2. Run the executable:
   ./prog

## How to Play

- Players take turns entering a number between 1 and 9.
- The number corresponds to the position on the board.
- First player to align 3 symbols wins.
- If all 9 moves are used without a winner, the game ends in a draw.
